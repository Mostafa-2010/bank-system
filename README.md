# bank-system
A .py program that simulates a real bank program with working features. It can create account, deposit, withdraw, delete account and more developer options and commands.

    • Features:
        ○ Create account with name, passcode, and balance (0 $)
        ○ Delete account with your passcode.○ 
        ○ Find an account and enter with your passcode. 
        ○ Change passcode.
        ○ Show transactions history. 
        ○ Delete transaction history.
        ○ Deposit and withdraw.
        ○ Show balance.
        ○ Clear all data with dev passcode. (DEV ACCESS ONLY)
        ○ Show all data with dev passcode.  (DEV ACCESS ONLY)

    • How to use:
        ○ Run main.py.
